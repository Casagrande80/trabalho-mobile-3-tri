## 🛠️ INSTRUÇÕES FINAIS DE USO

### Para usar o projeto IMEDIATAMENTE (sem API):
1. Crie a pasta `projeto-restaurantes`
2. Cole todos os arquivos HTML na pasta raiz
3. Crie as pastas `css`, `js`, `assets/images`
4. Cole os arquivos correspondentes em cada pasta
5. Abra `login.html` no navegador

### Para usar com API (recomendado):
1. Siga os passos acima
2. Instale o JSON Server: `npm install -g json-server`
3. Execute: `json-server --watch db.json --port 3000`
4. Abra `login.html`

### Credenciais de login (simuladas):
- **Email**: qualquer email válido
- **Senha**: qualquer senha (o sistema aceita qualquer coisa)
